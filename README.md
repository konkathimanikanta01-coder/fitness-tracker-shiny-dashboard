## Project Overview
This project is an interactive fitness tracking dashboard developed using R Shiny. It enables users to analyse activity data such as steps, calories burned, distance and activity trends through dynamic and interactive visualisations.

The objective is to demonstrate dashboard development, data visualisation and user-focused analytics for decision support.

## Features
- Interactive Shiny dashboard
- Activity trend analysis
- Calories, distance and step tracking
- Dynamic filtering and visualisation
- User-friendly interface

## Tools & Technologies
R | Shiny | ggplot2 | dplyr | DT | lubridate | Data Visualisation

## How to Run Locally
Detailed instructions are provided in:
How_to_run_app_Locally.txt
1. Download repository
2. Open app.R in RStudio
3. Install packages:

install.packages(c("shiny","ggplot2","dplyr","DT","shinyBS","shinycssloaders","lubridate"))

4. Run:
shiny::runApp()

Ensure dataset is in same folder as app.R.
